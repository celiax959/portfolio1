Installation Service

Administration de systèmes communicants complexes avec services interconnectés

Description du projet

Pour la validation de la compétence 3 du deuxième semestre, nous avons installé un service de base de données, un service http de pages web dynamiques et un service de messagerie dans des machines virtuelles qui communiquaient entre elles. Nous avons également rédigé une notice d'installation de chaque service.

Nous devions vérifier que chaque machine fonctionnait à l'aide de tests comme envoyer un mail fictif depuis un service de messagerie ou rechercher notre page php sur un navigateur à l'aide de l'adresse ip de la machine virtuelle.

Cette SAE nous a permis de commencer à se familiariser au langage php pour la suite du BUT.
