Installation de Poste

Dual boot Linux/Windows et machines virtuelles

Description du projet

Pour la validation de la compétence 3 du premier semestre, nous avons installé, en binôme, un dual boot Linux/Windows ainsi qu'une machine virtuelle sur un ordinateur.

Nous avons également rédigé une notice d'utilisation présentant les avantages, les inconvénients et les cas d'usage de chaque méthode.

Objectifs du projet

Cette SAE nous a été très utile, car en BUT informatique, il est essentiel d'avoir accès aux systèmes d'exploitation Linux et Windows pour travailler efficacement. Le projet nous a permis de :

  • Comprendre les différences entre les systèmes d'exploitation
  • Maîtriser l'installation et la configuration de Linux
  • Apprendre à utiliser les machines virtuelles
  • Rédiger une documentation technique complète

Réalisation technique

Nous avons configuré :

  • Dual boot Windows 10 / Ubuntu Linux
  • Machine virtuelle avec VirtualBox
  • Partage de fichiers entre systèmes
  • Configuration réseau pour les machines virtuelles

Compétences développées

  • Administration système Linux/Windows
  • Gestion de partitions et bootloader
  • Virtualisation avec VirtualBox
  • Rédaction de documentation technique
  • Résolution de problèmes matériels
