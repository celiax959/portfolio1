Escape Game Vice Versa

Site web promotionnel pour un escape game sur le thème du film

Description du projet

Pour la validation de la compétence 5 du premier semestre, nous avons créé un site internet présentant un Escape Game. Notre groupe a choisi le thème de Vice Versa 2, permettant aux visiteurs de plonger dans l'univers des émotions.

Concept et design

Le site propose une immersion dans le Royaume des Émotions avec des salles thématiques représentant différentes émotions. Chaque salle a été conçue avec soin pour offrir une expérience unique, avec des décors époustouflants et des défis adaptés.

Le site inclut :

  • Présentation détaillée de l'escape game
  • Gallerie d'images des différentes salles
  • Système de réservation en ligne
  • Informations pratiques et tarifs
  • Section "Qui sommes-nous"

Technologies utilisées

  • HTML5 et CSS3
  • JavaScript pour les interactions
  • Design responsive
  • Intégration de formulaires
